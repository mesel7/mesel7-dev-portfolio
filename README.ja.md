# mesel7 Dev Portfolio

ReactとFirebaseを基盤に制作した個人ポートフォリオサイトです。 
爽やかな夏の空から着想を得たイメージのカラーパレットを採用し、プロジェクト・経験・技術スタックを体系的に整理して、就職活動や自己PRに活用できるよう構成しました。

---

## 🧭 プロジェクト概要

- **期間:** 2024.07 ~ 2025.08 
- **タイプ:** Personal Project / Fullstack
- **担当:** 全機能の設計・フロントエンド実装、Firebaseを用いたバックエンド構築およびデプロイ

---

## 🛠 技術スタック

| 区分 | 技術スタック |
|------|------|
| Frontend | HTML5, CSS3, JavaScript, React |
| Infra & Deployment | Firebase Hosting |
| Collaboration & Versioning | GitHub |

---

## 🎯 主な機能

- 自己紹介: 紹介文とプロフィールを通じて、訪問者の方が分かりやすくご覧いただけるように構成いたしました。  
- 技術スタック: 保有している主要な技術スキルを簡潔に整理いたしました。  
- 開発経験: 主なプロジェクトと開発過程を要約してご紹介いたしました。  
- プロジェクト: 個人およびチームのプロジェクトをカード型の一覧としてご提供いたしました。  
- プロジェクト詳細: 各プロジェクトの背景、課題解決の過程、学んだ点を詳しく記録いたしました。  
- 多言語対応: i18nライブラリを活用し、日本語と韓国語の切り替え機能を実装いたしました。
